# PythonLearn

プログラミング完全初心者向けの、ブラウザで学べる日本語Python学習サイトです。PythonコードはPyodideによりブラウザ内のWeb Workerで実行されます。サーバーやファイルにコードが送信されることはありません。

## できること

- 全10章・43レッスンの解説、実行例、演習
- CodeMirror 6のPythonエディタ（CDNが使えない場合はテキストエリアに自動切替）
- 演習の自動判定と、localStorageへの合格記録保存
- 無限ループなどを10秒で停止し、実行環境を再生成
- PC・タブレット・スマートフォン向けレスポンシブ表示

## ローカルで確認する

`file://` で直接開かず、リポジトリのルートでHTTPサーバーを起動してください。

```bash
python -m http.server 8000
```

ブラウザで <http://localhost:8000/> を開きます。終了はターミナルで `Ctrl+C` です。

初回の実行時はPyodide本体をCDNから読み込むため、ネットワーク接続と少しの待ち時間が必要です。

## 演習データの検証

次のコマンドは、全演習で「模範解答は合格」「雛形は不合格」となることを確認します。

```bash
python tests/validate_lessons.py
```

## GitHub Pagesへの公開

ビルドは不要です。GitHubリポジトリのPages設定で、このリポジトリのブランチとルートディレクトリを公開元に指定します。すべての内部パスは相対パスなので、`https://<user>.github.io/<repository>/` のようなサブパス配下でも動作します。

## 注意

- `input()` は使えません。入力値は変数に代入して与えてください。
- 外部CDN（Pyodide・CodeMirror）へ接続できない環境では、実行またはエディタ強調表示を利用できません。

LESSON 01

画面に「こんにちは」
と表示してみよう

プログラムはコンピュータへの指示書です。`Console.WriteLine` は、その指示を画面に表示します。

このページはC#コースの構成案です。コード実行機能は、次の段階で追加します。

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("こんにちは、C#!");
    }
}
